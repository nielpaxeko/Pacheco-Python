CSS Recap: Basic Concepts

1. How to Add CSS

There are three main ways to apply CSS to an HTML document:

Inline CSS: Applied directly to an HTML element using the style attribute.

<p style="color: blue; font-size: 16px;">This is a paragraph.</p>

Internal CSS: Defined within a <style> tag inside the <head> of an HTML document.

<style>
  p {
    color: blue;
    font-size: 16px;
  }
</style>

External CSS: Defined in an external file (styles.css) and linked using the <link> tag.

<link rel="stylesheet" href="styles.css">

2. CSS Selectors

Selectors are used to target HTML elements.

Universal Selector (*): Targets all elements.

* {
  margin: 0;
  padding: 0;
}

Element Selector: Targets specific HTML tags.

p {
  color: red;
}

Class Selector (.): Targets elements with a specific class.

.highlight {
  background-color: yellow;
}

ID Selector (#): Targets an element with a specific ID.

#header {
  font-size: 24px;
}

Group Selector (A, B): Styles multiple elements.

h1, h2, h3 {
  color: navy;
}

Descendant Selector (A B): Targets elements inside another element.

div p {
  color: green;
}

3. Colors in CSS

Colors can be defined in different ways:

Named Colors: red, blue, green, etc.

Hexadecimal (#RRGGBB): #ff0000 (red), #00ff00 (green)

RGB (rgb(r, g, b)): rgb(255, 0, 0) (red)

RGBA (rgba(r, g, b, a)): rgba(255, 0, 0, 0.5) (50% transparent red)

HSL (hsl(h, s%, l%)): hsl(0, 100%, 50%) (red)

4. Fonts in CSS

Font Family: Defines the text font.

p {
  font-family: Arial, sans-serif;
}

Font Size: Defines text size.

p {
  font-size: 16px;
}

Font Weight: Controls boldness (normal, bold, lighter, bolder, 100-900).

p {
  font-weight: bold;
}

Text Alignment: left, right, center, justify.

p {
  text-align: center;
}

5. CSS Box Model

Each element is a rectangular box consisting of:

Content: The actual content (text, image, etc.).

Padding: Space around the content (inside the border).

Border: Surrounds the padding.

Margin: Space outside the border (between elements).

div {
  width: 200px;
  padding: 10px;
  border: 5px solid black;
  margin: 20px;
}

6. Other Basic CSS Knowledge

Display Property: Controls element display (block, inline, flex, grid, none).

span {
  display: block;
}

Positioning:

static (default)

relative (relative to itself)

absolute (relative to nearest positioned ancestor)

fixed (relative to viewport)

div {
  position: absolute;
  top: 50px;
  left: 100px;
}

Flexbox (for layout control):

.container {
  display: flex;
  justify-content: space-between;
}
